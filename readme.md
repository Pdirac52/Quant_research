# 主动买卖资金流因子研究

本项目用于基于日频股票数据构建主动买卖资金流因子，并完成因子有效性分析。当前数据字段包括收益率、主动买入金额、主动卖出金额等。

## 项目目标

- 构造主动买卖资金流指标 `a`
- 基于过去 20 个交易日中的低收益日计算因子值
- 完成因子 IC 分析
- 绘制分组收益图与多空收益曲线

## 因子定义

主动买卖资金流指标：

```text
a = (主动买入金额 - 主动卖出金额) / (主动买入金额 + 主动卖出金额)
```

因子计算逻辑：

```text
lambda = 0.2
回溯过去 20 个交易日
选取收益率最低的 lambda 比例交易日
对这些低收益日的 a 取平均，得到 factor
```

## 已完成内容

### 模块一：基础数据体检

已完成原始数据读取与基础结构检查，主要包括：

- 读取 CSV 数据
- 将 `date` 字段转换为 datetime 类型
- 按 `order_book_id` 和 `date` 排序
- 查看数据维度、字段名和字段类型
- 统计股票数量、交易日数量和日期范围
- 检查面板数据完整性
- 统计每只股票的数据覆盖情况
- 检查 `(order_book_id, date)` 是否存在重复主键

该模块的目的是在正式计算因子前确认数据结构是否可靠，避免 rolling、shift 等时序操作出现隐性错误。

### 模块二：整体 NaN 统计

已完成核心字段缺失值统计，主要包括：

- 统计各字段 NaN 数量和比例
- 重点检查 `buy_value`、`sell_value`、`pct_rate`
- 按股票统计核心字段 NaN 比例
- 按日期统计核心字段 NaN 比例
- 识别核心字段严重缺失的异常交易日

该模块的目的是区分不同类型的缺失值，避免将结构性缺失、停牌缺失和数据源缺失混为一类处理。
